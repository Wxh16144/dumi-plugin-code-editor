# dumi-plugin-code-editor

> 将代码块转换为**只读**代码编辑器, See [#2175](https://github.com/umijs/dumi/issues/2175)

## Usage

### Install

```bash
npm install dumi-plugin-code-editor --save-dev
```

### Apply

```ts
// .dumirc.ts
export default {
  plugins: ['dumi-plugin-code-editor'],
};
```
